You will get token smart contract, presale token smart contract. So, you can create token contract and presale contract for your own custom token.

You can watch video tutorial for following steps.

Create contract files using Remix IDE.

Firstly, you have to go using web browser to https://remix.ethereum.org/ . Create new files. Copy token contract and presale contract from provided folder and paste it on new created files in remix IDE.

Compile contract

You have to compile contract before deploying it. Go to compiler tab section. Select the latest compiler version.

Click over Compile button. Then, Contract will be compiled.

You have to keep abi json for interacting with smart contract using react app.

Deploy Contract

Go to Deploy tab section. Select injected provider metamask for connecting metamask wallet into remix IDE.

You have to select desired network for deploying contract. You can select any network from avalable list.

Deploy token smart contract.

Click over Deploy button. Then, you have to click over Confirm button in metamask popup.

Customize presale smart contract before deploying

You have to customize provided presale smart contract file before deploying on blockchain. You can add presale token limit inside presale contract.

You can add stage information before deploying presale contract. To add stage information just follow the following image.

Deploy presale smart contract

You have to set payment address, token address, price feed address before deploying presale smart contract.

In _payment field, paste payment address(token purchased price goes to payment address directly).

In _token field, paste token contract address.

In _pricefeed field, paste price feed address(use following address or contact with us).

Transfer token for presale contract

You can give some tokens to presale contract for selling. There are two ways you can do it. You can mint some tokens for presale or can transfer some tokens to presale. So, you can execute mint or transfer.

You have to do this step inside token contract. Click transfer function.
Inside "to" field, paste presale token contract address.
Inside "amount" field, paste token amount(wei format) to transfer.
Then have to click over transact button. Finaly, click over Confirm button in metamask popup. Then token amount will be transfered for presale contract.

You can use another step for transfering token also inside token contract. Click mint function.
Inside "to" field, paste presale token contract address.
Inside "amount" field, paste token amount(wei format) to transfer.
Then have to click over transact button. Finaly, click over Confirm button in metamask popup. Then token amount will be minted for presale contract.

Add stage information in presale contract

You can add more stage information following this steps.
You have to fill stage information properly, then click over transact button. Finaly, click over Confirm button in metamask popup. Repeat this step to add more stage in presale contract.

In _bonus field, paste bonus amount between 0 to 100.

In _price field, paste price amount(wei format) for 1 token. eg. if you would like to set 1 token = 0.001 USD, so you have to paste 1000000000000000(wei format for 0.001) inside _price field.

In _start field, paste start time(unix timestamp).

In _end field, paste end time(unix timestamp).

Contract Configuration

You can watch video instruction for interacting with smart contract using this template.

Add contract in react app

Go to "config.js" file. Paste token contract address for "tokenContractAddress" variable. Paste presale token contract address for "presaleContractAddress" variable.

Add ABI json of contracts

You have already deployed token contract and presale token contract. So, you have kept abi json of those contracts. Now you have to add abi json into react app. Go to "TokenContractAbi.json" and paste token contract abi json. Go to "PresaleContractAbi.json" and paste token contract abi json.

Import ABI json file

Go to "config.js" file. Import "TokenContractAbi.json" and "PresaleContractAbi.json" file.

Interacting with smart contract

Go to "config.js" file.

If you want to customize according to own requirement, you will do it . For this you have to use wagmi. Using it you can customize interation(read, write contract) with this theme.

go to "src/utils/PresaleContextProvider.jsx" file. Then you can understand the interaction of read and write contract function.
For interacting with smart contract you must have to knowledge about wagmi. So, you can understand and write code according to your custom requirement inside "src/utils/PresaleContextProvider.jsx" file.

Deploy

Gittu react version Deploy

Gittu React js static build

  • Navigate to gittu_react_main
  • You can type path url inside "vite-config.js" file. If you are deploying gittu react app to server you can type domain/custom path url inside base in "vite-config.js" file.
  • run command npm run build
  • after the build is done you will get a folder called dist.
  • now you can host that folder anywhere you want (If you are deploying gittu react app to server, you have to type domain/custom path url inside base in "vite-config.js" file before build).
  • For more information you can visit Deploying a Static Site

netlify

  • Navigate to gittu_react_main
  • run command:
  • npm run build

    after the build process is done. you will find a new folder called /dist

  • npm install netlify-cli --save-dev
  • netlify login

    after the login process is done

  • run command: netlify deploy
  • choose + Create & configure a new site
  • give you site name
  • path to deploy: ./dist

    after the build process once done, run command:

  • netlify deploy --prod
  • path to deploy: ./dist
  • for more information: How to deploy React Apps in netlify

You will find react.js app deployment complete guide here: React.js app Deployment
